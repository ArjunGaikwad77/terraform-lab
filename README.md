# Terraform Lab

This repository contains assignments for learning Infrastructure as Code using Terraform.

Structure:

terraform-lab/
├── day1
├── day2
├── day3
├── day4
├── day5
└── modules

Each directory demonstrates different Terraform concepts:
- Day1: Basic Terraform workflow
- Day2: Variables and tfvars
- Day3: Docker container management
- Day4: Terraform modules
- Day5: CI automation with GitHub Actions
